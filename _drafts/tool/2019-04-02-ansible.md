### 配置

* 全局配置：`/etc/ansible`
* 局部配置：在当前项目里编辑配置文件

部署文件：

    - deploy-project
        - ansible.cfg
        - deploy.yml
        - hosts

ansible.cfg

``` cfg
[defaults]
# 主机清单
inventory = hosts
# 远程登录的用户名
remote_user = root
# 用于 ssh 登录的私钥文件
private_key_file = /root/.ssh/id_rsa
host_key_checking = False
```
