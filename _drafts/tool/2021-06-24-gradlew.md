# Gradle

## 配置文件

* `settings.gradle`：项目的根配置
* `build.gradle`：项目的构建脚本，包含任务、依赖、插件
* `gradle.properties`：项目或全局范围的属性
* `gradlew`：gradle 包装器脚本
* `gradle` 目录
    * `wrapper` 目录
        * `gradle-wrapper.properties`：配置 gradle 包装器的属性，如指定 gradle 版本和下载 url
    * `dependencies.gradle`

## 构建的生命周期

### Initialization

配置构建环境以及有那些 Project 会参与构建（ 解析 `settings.build`）

### Configuration

生成参与构建的 Task 的有向无环图以及执行属于配置阶段的代码( 解析 `build.gradle` )

### Execution

按序执行所有 Task
