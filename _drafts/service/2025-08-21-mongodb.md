# MongoDB

## 概念

- 数据库 Database -- 多个 Collection 的容器
- 集合 Collection -- 类似关系型数据库的 表 Table
- 文档 Document   -- 类似关系型数据库的 行 Row，但结构可变
- 字段 Field      -- 类似关系型数据库的 列 Colume
- 索引 Index      -- 用于加速查询

## 架构

### 副本集 Replica Set

分为 Primary 节点与 Secondary 节点，所有节点保存同一份完整数据，Secondary 通过 Oplog 从 Primary 复制。

Primary 故障时，Secondary 可选举出新的 Primary。

### 分片集群 Sharded Cluster

水平扩展存储与吞吐，按分片键 shard key 将集合切分为 chunk，分布到多个分片上。

每个分片自身通常是一个副本集 Replica Set。

分片集群前会有多个 Mongos 作为代理，负责接收请求，将不同的数据分发到不同的分片上。

Config Server 存储数据分布信息。


## 操作


### mongosh

https://www.mongodb.com/try/download/shell


以 ubuntu 24.04 为例，下载并安装 mongosh

    wget https://downloads.mongodb.com/compass/mongodb-mongosh_1.10.6_amd64.deb

    sudo apt install -y ./mongodb-mongosh_1.10.6_amd64.deb

### 连接服务

本地服务：

    mongosh

远程服务：

    mongosh "mongodb://user:pass@host1:27017,host2:27017/dbname?replicaSet=rs0"

### 数据库

查看当前数据库

    db

切换/创建数据库（不存在会自动创建）

    use mydb

查看所有数据库

    show dbs

### 集合

创建集合

    db.createCollection("users", {validator: {$jsonSchema: {bsonType: "object", required: ["name"]}}})

删除集合

    db.users.drop()

单条插入

    db.users.insertOne({name: "Alice", age: 28, password: "xxxx"})

多条插入

    db.users.insertMany([{name: "Bob", password: "yyy"}, {name: "Carol", age: 32, password: "zzz"}])

查询

    db.users.find()

条件查询

    db.users.find({age: {$gt: 20, $lte: 40}})

单条更新

    db.users.updateOne({name: "Alice"}, {$set: {age: 29}})

多条更新

    db.users.updateMany({age: {$lt: 18}}, {$set: {minor: true}})

单条删除

    db.users.deleteOne({name: "Bob"})

多条删除

    db.users.deleteMany({age: { $lt: 18}})

### 副本集
